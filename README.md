Batiyana 🛒🎙️

A voice-first grocery shopping app — think Blinkit, but you can just batiyana (chat) with your list.
Say "Add milk and bread to my list" or "Place my order," and Batiyana handles the rest.

✨ Features


Voice commands — add/remove items, check your list, place orders, and navigate the app hands-free using the Web Speech API
Smart list management — track quantities, mark items done, see live progress toward checkout
Daily Essentials catalog — 50+ everyday grocery items across categories (Dairy, Fruits, Vegetables, Staples, Beverages, Snacks, Household, Spices) with category filters
Personalized suggestions — smart swaps, frequently bought items, and flash deals
One-tap checkout — a sticky Place Order bar summarizes your cart and submits the order
Command history & confirmation flow — risky actions (like placing an order) ask for confirmation before executing, with a running log of recent voice commands
